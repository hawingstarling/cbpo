# Project Overview

## Name
plat-pf-web

## Description
Vue 2.6 app with CoreUI/BootstrapVue, Vuex, Router.

## Technology Stack
- Vue 2.6.10, Vuex 3.0.1, Vue Router 3.0.3
- CoreUI 2.1.9, BootstrapVue 2.0.4
- SCSS, Axios

## Repository Structure
- src/components, src/store, src/router, src/shared, src/mixins

## Getting Started
- npm install && npm run serve
